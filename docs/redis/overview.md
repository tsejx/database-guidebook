# Radis

```bash
# 启动
redis-server

# 测试
redis-cli

# 设置 key 值
set KEY VALUE

# 获取 key 值
get KEY

# 查看 key 是否存在
exists key

# 查看所有的 key
keys *

# 消除所有的 key
flushall
```
