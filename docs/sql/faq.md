## 无法连接本地数据库服务器

```bash
ERROR 2002 (HY000): Can't connect to local MySQL server through socket '/var/lib/mysql/mysql.sock'
```

意味着：

1. MySQL 服务器未安装或未运行
2. `mysql.sock` 不存在于 `/var/lib/mysql/`
